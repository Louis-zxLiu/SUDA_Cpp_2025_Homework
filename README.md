## 📂 项目结构

本项目包含苏州大学2024级C++课程的所有实验报告和源代码，主要分为两个部分：

## 🚀 如何运行

本项目代码主要使用 VS Code 编写，每个源代码目录下包含 `.vscode` 配置文件夹。

1.  **环境要求**:
    *   C++ 编译器 (如 MinGW/GCC)
    *   Visual Studio Code (推荐)

2.  **编译运行**:
    *   可以直接在 VS Code 中打开对应的文件夹，使用配置好的 Task 进行编译运行。
    *   或者使用命令行编译：
        ```bash
        g++ filename.cpp -o filename.exe
        ./filename.exe
        ```

## ⚠️ 注意事项

*   部分代码中使用了绝对路径（如 `c:\\data\\data.txt`），在运行 Class8 相关代码时，请根据你的实际路径修改代码或创建对应的文件目录。
*   `.exe` 文件为编译后的可执行文件，建议重新编译以确保在你的环境中正常运行。
